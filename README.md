# AI Email Sender (Python)

Program Python sederhana untuk **mengirim email otomatis menggunakan SMTP**, dilengkapi dengan **logika AI sederhana (rule-based)**. Cocok untuk belajar automation, dasar AI, dan project portfolio.

---

## Fitur

* Kirim email otomatis menggunakan Python
* Login aman menggunakan **App Password Gmail**
* Support subject & isi email custom
* Bisa dikembangkan menjadi:

  * Email berbasis jam
  * Email berbasis input / kata kunci
  * Email massal (CSV)
  * Email bot CLI

---

## Teknologi

* Python 3.x
* `smtplib`
* `email.message`

(Tidak menggunakan library eksternal)
